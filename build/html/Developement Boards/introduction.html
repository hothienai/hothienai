Development Boards

Introduction

This section provides practical guidance on working with popular embedded development boards. You’ll learn about board architectures, development environments, toolchains, and how to get started with hands-on embedded programming projects using real hardware.

Topics to be covered

  1. Introduction to Development Boards - What is a development board? - Development board vs production hardware - Evaluation boards vs learning platforms - Selecting the right board for your project

  2. Popular Microcontroller Families - ARM Cortex-M series (STM32, NXP, etc.) - AVR (Arduino, ATmega) - PIC microcontrollers - ESP32/ESP8266 (Wi-Fi enabled) - Raspberry Pi Pico (RP2040)

  3. Arduino Platform - Arduino Uno, Nano, Mega - Arduino IDE setup - Arduino libraries and ecosystem - Arduino vs bare-metal programming - Uploading and debugging

  4. STM32 Development Boards - STM32 Nucleo boards - STM32 Discovery boards - STM32CubeIDE setup - HAL vs LL libraries - STM32CubeMX configuration tool

  5. ESP32/ESP8266 Boards - NodeMCU and development boards - ESP-IDF vs Arduino framework - Wi-Fi and Bluetooth capabilities - IoT applications - OTA (Over-The-Air) updates

  6. Raspberry Pi Pico - RP2040 microcontroller features - MicroPython vs C/C++ SDK - PIO (Programmable I/O) - Dual-core programming - Development environment setup

  7. Development Environment Setup - IDE selection (VS Code, Eclipse, Keil, IAR) - Compiler toolchains (GCC, Clang) - Build systems (Make, CMake) - Version control integration - Debugging tools

  8. Programming and Flashing - USB-to-serial converters - ST-Link, J-Link programmers - JTAG and SWD interfaces - Bootloaders - Flash memory programming

  9. Board Features and Peripherals - On-board LEDs and buttons - USB connectivity - Serial communication (UART) - Power supply options - Clock sources (crystals, oscillators)

  10. Debugging Hardware - On-board debuggers - External debug probes - Serial wire debug (SWD) - Real-time debugging - Breakpoints and watchpoints

  11. Shields and Expansion Boards - Arduino shields - STM32 expansion boards (X-NUCLEO) - Sensor modules - Display modules - Motor driver shields

  12. Power Management - USB power vs external power - Voltage regulators - Battery operation - Power consumption measurement - Low-power modes

  13. Pin Configuration - GPIO pin mapping - Pin multiplexing (alternate functions) - Voltage levels (3.3V vs 5V) - Pin protection and limitations - Breadboard connections

  14. Example Projects - Blinking LED (Hello World) - Button input and debouncing - UART communication - ADC reading (sensors) - PWM control (LED dimming, servo)

  15. Communication Interfaces - I2C device connection - SPI peripheral interfacing - UART console and debugging - USB CDC (Virtual COM Port) - CAN bus (if available)

  16. Sensor Integration - Temperature and humidity sensors - Motion sensors (accelerometer, gyroscope) - Light sensors - Distance sensors (ultrasonic, IR) - Sensor libraries and drivers

  17. Display Integration - LCD character displays - OLED displays - TFT color displays - 7-segment displays - Graphics libraries

  18. Wireless Communication - Wi-Fi modules - Bluetooth (Classic and BLE) - LoRa/LoRaWAN - NFC/RFID readers - Radio modules (nRF24L01)

  19. Prototyping and Testing - Breadboard prototyping - Jumper wires and connections - Multimeter measurements - Logic analyzer usage - Oscilloscope basics

  20. Best Practices - Hardware handling and care - ESD protection - Proper connections and wiring - Power supply considerations - Documentation and schematics - Troubleshooting common issues
